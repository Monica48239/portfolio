# 学歴職歴管理システム

## 概要
UdemyのSpring Bootコースで学習中のWebアプリです。  
データベース操作を使った学歴・職歴の管理システムを開発しています。

## 学習目的
- Spring Bootを使ったWebアプリ開発の習得
- JDBCを使ったデータベース操作の実装
- CRUD（登録・参照・更新・削除）の実装

## 使用技術
- Java
- Spring Boot
- JDBC
- H2DB
- Thymeleaf
- HTML

## 実装済み
- Controller（MemberController・ResumeController）
- 画面（HTML）：一覧・登録・編集画面
- schema.sql・application.properties
